│                                                             │
│   🔌 Install Extension                                      │

```

### 2. Automatic Privacy Policy Detection

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   🔍 Automatic Detection                                    │
│   ────────────────────                                      │
│                                                             │
│   When you visit a privacy policy page:                     │
│                                                             │
│   1. A notification will appear                             │
│   2. Click "Analyze Now" to analyze the policy              │
│   3. View results in the popup                              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3. Manual Analysis

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   🖱️ Manual Analysis                                        │
│   ───────────────                                           │
│                                                             │
│   1. Click the LegalLens icon in your browser toolbar       │
│   2. Select "Extract from Current Page"                     │
│   3. Review the extracted text                              │
│   4. Click "Analyze" to process the text                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Troubleshooting

### 1. Common Issues

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   🔧 Common Issues                                          │
│   ─────────────                                             │
│                                                             │
│   Issue: Analysis fails to complete                         │
│   Solution: Try with a smaller text sample or refresh       │
